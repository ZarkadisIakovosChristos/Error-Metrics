# Error-Metrics
This repository contains information and material for the Error-Metrics, a personal project. The purpose of the project is to create error metrics for regresssion, calssification and clustering tasks, for a more compact usage. The source files are written in Python and R. Some scores included are R2, OOS-R2, MSE, MSR, SSE, SSR, MAE, MAPE, SMAPE, AIC, BIC, Accuracy, F1, Precision, Recall, Roc-Auc.
